# Calculator
This is a basic calculator on c++
## supported operations
* '/' **for division**
* '+' **for add**
* '-' **for subtraction**
* '*' **for multiply**
## Features:
* Infinite loop until you type `quit`.
* Protection against division by zero.
* Fast and lightweight.

## How to start
* 1 go to releases
* 2 select version and download file
